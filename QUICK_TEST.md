## 🎯 ЧТО ПРОВЕРИТЬ

### У уволенных сотрудников:
- [ ] Тег 🔴 **"Уволен"** с красным фоном (НЕ "Не принят")
- [ ] ПИНФЛ `71112 94123 4571` - не подтвержден + уволен
- [ ] ПИНФЛ `80507 95123 4572` - подтвержден + уволен

### Конфликт слияния:
- [ ] ПИНФЛ `90808 96123 4573`
- [ ] Показываются **ДВЕ карточки**: локальная + госбаза
- [ ] Разные данные: отчество и паспорт
- [ ] Радиокнопки для выбора
- [ ] Кнопка "Объединить записи"

### Смарт-панель:
- [ ] Принят на работу → только "Сохранить"
- [ ] Не принят → "Принять на работу" + "ГПХ" + "Сохранить"
- [ ] Уволен → "Принять заново" + "ГПХ"
