# 🧪 ШПАРГАЛКА: Тестовые номера ПИНФЛ

## 📋 Быстрый доступ к номерам для тестирования

---

## 1️⃣ НОВЫЙ СОТРУДНИК (найден в госбазе, не в локальной)

```
✅ 12345 67890 1234  →  Ахмедов Азиз Рашидович (Ташкент)
✅ 98765 43210 9876  →  Каримова Нигора Алишеровна (Самарканд)
```

**Ожидаемый результат:**
- Статус: "Не принят" (зеленый)
- Данные подтверждены ✅
- Смарт-панель: "Принять на работу" + "ГПХ" + "Сохранить"

---

## 2️⃣ НЕ НАЙДЕН В ГОСБАЗЕ

```
❌ 11111 11111 1111  →  Любой номер, которого нет в базе
```

**Ожидаемый результат:**
- Alert: "Не найдено в госбазе"
- Кнопка: "Добавить вручную"
- Квота: НЕ тратится

---

## 3️⃣ ОШИБКА API/СЕРВЕРА

```
🔴 88888 88888 8888  →  Тест ошибки сервера
```

**Ожидаемый результат:**
- Alert: "Ошибка при обращении к госбазе"
- Квота: -1 (тратится!)

---

## 4️⃣ НЕТ СОЕДИНЕНИЯ

```
📡 77777 77777 7777  →  Тест потери связи
```

**Ожидаемый результат:**
- Alert: "Нет соединения с сервером"
- Квота: НЕ тратится

---

## 5️⃣ ДУБЛИКАТ: Не подтвержден + Принят

```
🟡 30001 85123 4567  →  Иванов Иван Иванович
```

**Ожидаемый результат:**
- isVerified: ❌ false
- isHired: ✅ true
- Кнопка под карточкой: "Обновить данные из госбазы" 🔄
- Смарт-панель: ТОЛЬКО "Сохранить без оформления"

---

## 6️⃣ ДУБЛИКАТ: Подтвержден + Принят

```
✅ 40101 90123 4568  →  Петров Петр Петрович
```

**Ожидаемый результат:**
- isVerified: ✅ true (01.03.2026)
- isHired: ✅ true
- Статус: "Принят"
- Смарт-панель: ТОЛЬКО "Сохранить без оформления"

---

## 7️⃣ ДУБЛИКАТ: Не подтвержден + НЕ принят

```
🟡 50203 92123 4569  →  Сидорова Мария Викторовна
```

**Ожидаемый результат:**
- isVerified: ❌ false
- isHired: ❌ false
- Статус: "Не принят"
- Кнопка под карточкой: "Обновить данные из госбазы" 🔄
- Смарт-панель: "Принять на работу" + "ГПХ" + "Сохранить"

---

## 8️⃣ ДУБЛИКАТ: Подтвержден + НЕ принят

```
✅ 60305 93123 4570  →  Каримов Алишер Рахимович
```

**Ожидаемый результат:**
- isVerified: ✅ true (10.03.2026)
- isHired: ❌ false
- Статус: "Не принят"
- Смарт-панель: "Принять на работу" + "ГПХ" + "Сохранить"

---

## 9️⃣ ДУБЛИКАТ: Не подтвержден + УВОЛЕН

```

```

**Ожидаемый результат:**
- isVerified: ✅ true (15.02.2026)
- employmentStatus: 🔴 terminated

---

## 1️⃣1️⃣ КОНФЛИКТ СЛИЯНИЯ (данные расходятся)

```
⚠️ 90808 96123 4573  →  Рахимов Тимур (конфликт данных)
```

**Ожидаемый результат:**
- ⚠️ Toast: "Обнаружен конфликт данных"
- 📊 Показываются ДВЕ карточки:
  - **Локальная запись:** Рахимов Тимур Олегович, паспорт MN 3334567
  - **Данные из госбазы:** Рахимов Тимур Шавкатович, паспорт MN 9998888
- 🔘 Нужно выбрать радиокнопку на одной из карточек
- 🔄 Кнопка "Объединить записи" (доступна только после выбора)
- 🎯 Смарт-панель: инструкция по разрешению конфликта
- ✅ Квота: -1

## 🎯 РЕКОМЕНДУЕМАЯ ПОСЛЕДОВАТЕЛЬНОСТЬ ТЕСТИРОВАНИЯ

### Этап 1: Базовые сценарии
1. `12345 67890 1234` - новый сотрудник
2. `11111 11111 1111` - не найден
3. `88888 88888 8888` - ошибка API
4. `77777 77777 7777` - нет связи

### Этап 2: Дубликаты (не подтверждены)
5. `30001 85123 4567` - не подтвержден + принят
6. `50203 92123 4569` - не подтвержден + не принят
7. `71112 94123 4571` - не подтвержден + уволен

### Этап 3: Дубликаты (подтверждены)
8. `40101 90123 4568` - подтвержден + принят
9. `60305 93123 4570` - подтвержден + не принят
10. `80507 95123 4572` - подтвержден + уволен

### Этап 4: Конфликты
11. `90808 96123 4573` - конфликт данных (выбор записи)
